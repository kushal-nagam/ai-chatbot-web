# AI Chatbot Web App

This is a web-based AI chatbot built using Streamlit and OpenAI API.

## Features
- Interactive UI
- Real-time chatbot
- Simple web interface

## Tech Stack
- Python
- Streamlit
- OpenAI API

## Run Locally
pip install -r requirements.txt  
streamlit run app.py
